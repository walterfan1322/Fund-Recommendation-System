# 🤖 基金與股票個人化推薦系統

使用 **雙塔深度學習模型 (Dual-Tower)** 結合 **Yahoo Finance 即時數據**的智能推薦系統。

## ✨ 功能特色

- 📊 **個人化推薦**：基於年齡、風險偏好、投資目標等特徵
- 🔍 **即時數據**：從 Yahoo Finance 抓取最新市場數據
- 🤖 **AI 訓練**：使用 PyTorch 雙塔模型即時訓練
- 📈 **視覺化分析**：風險報酬圖、歷史走勢比較
- 💡 **智能解釋**：提供推薦理由

## 🚀 使用方式

1. **設定投資檔案**（左側欄）：輸入年齡、收入、風險偏好等資訊
2. **下載市場數據**：點擊按鈕從 Yahoo Finance 獲取最新數據
3. **選擇感興趣標的**：勾選您持有或關注的股票/基金
4. **訓練模型**：點擊「開始訓練」，AI 會學習您的偏好
5. **查看推薦**：獲得 Top-5 個人化推薦及詳細分析

## 🛠️ 技術棧

- **深度學習**：PyTorch 2.2.2
- **前端框架**：Streamlit
- **數據來源**：Yahoo Finance (yfinance)
- **視覺化**：Matplotlib, Plotly
